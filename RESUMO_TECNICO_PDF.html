VPO - Vemba Process Optimizer
Sistema Inteligente de Otimizacao de Separadores Trifasicos
Python / Streamlit | Versao 3.0 | Atualizado em 2026

Visao Geral

Sistema para otimizacao de separadores trifasicos (oleo/agua/gas) na industria de petroleo, implementado em Python com interface Streamlit. Integra modelagem fisica fundamentada em correlacoes empiricas reconhecidas, inteligencia artificial, otimizacao multiobjetivo e analise economica.

Nota importante: Os modelos fisicos sao baseados em correlacoes da literatura (Stokes, Souders-Brown, Beggs-Robinson). Os dados de treinamento da rede neural sao sinteticos. Para aplicacao industrial, e necessario calibrar os modelos com dados reais de planta (SCADA/PIMS).

Modelos Fisicos - Detalhamento

1. Eficiencia Gas-Liquido

Baseada no criterio de Souders-Brown para arraste de liquido e tempo de residencia.

Etapas do calculo:

  1. Densidade do gas corrigida por P e T (lei dos gases reais, Z=0.85):
    rho_gas = 1.2 * (P/10) * (288.15 / (T+273.15)) / 0.85
  2. Velocidade critica de Souders-Brown:
    v_crit = K * sqrt((rho_liq - rho_gas) / rho_gas), K = 0.107 - 0.0002*(P-10)
  3. Velocidade superficial do gas no separador:
    v_gas = (Q * GOR/1000) / (86400 * A_sep)
  4. Eficiencia por velocidade (sigmoidal):
    eff_v = 1 / (1 + exp(8 * (v_gas/v_crit - 0.7)))
  5. Eficiencia por residencia (exponencial, tau=3 min):
    eff_r = 1 - exp(-t_res / 3)
  6. Eficiencia total:
    eta_GL = 0.99 * eff_v * eff_r * min(eff_T, 1.02)

2. Eficiencia Oleo-Agua

Baseada na Lei de Stokes para sedimentacao gravitacional com correcao de viscosidade por Beggs-Robinson.

Etapas do calculo:

  1. Viscosidade corrigida por temperatura (Beggs-Robinson simplificada):
    mu_eff = mu * exp(2500 * (1/(T+273.15) - 1/333.15))
  2. Velocidade de Stokes para gota media d50=500um (com coalescedor):
    v_stokes = (2*g*(d/2)^2 * Delta_rho) / (9 * mu_eff * 1e-3)
  3. Razao de sedimentacao (residencia / sedimentacao):
    settling_ratio = t_res / (h_placa / v_stokes)

    h_placa = 0.15 m (distancia entre placas coalescedoras)

  4. Penalidade de emulsao (faixa critica 30-60% BSW):
    emulsion = 1 - 0.06 * exp(-((BSW/100 - 0.45) / 0.15)^2)
  5. Eficiencia total:
    eta_OA = 0.96 * eff_settling * emulsion

Defuzzificacao

Analises Complementares

  • Diagrama Tornado: Sensibilidade do VPL a variacao de +/-20% em cada componente
  • Cenarios: Otimista (+20% beneficios, -10% investimento), Realista (base), Pessimista (-30% beneficios, +15% investimento)
  • Monte Carlo: N simulacoes (100-1000) com incerteza configuravel (5-50%) nos beneficios e 10% no investimento. Calcula VPL medio, desvio padrao, probabilidade VPL>0, VaR 95%.

Limitacoes Conhecidas

  • Modelos nao calibrados com dados reais de planta
  • Rede neural treinada com dados sinteticos (R2 teste ~0.70)
  • Geometria do separador fixa (diametro 1.2m, volume 15m3)
  • Propriedades dos fluidos simplificadas (sem composicao detalhada)
  • Dados temporais do dashboard sao simulados

Conclusao

O VPO integra modelagem fisica fundamentada, inteligencia artificial e analise economica em uma interface unica para suporte a decisao na operacao de separadores trifasicos.

  • Modelagem fisica: Stokes, Souders-Brown, Beggs-Robinson, componentes de energia
  • Inteligencia artificial: MLPRegressor (128-64-32) com early stopping
  • Otimizacao: NSGA-II completo (SBX + mutacao polinomial + crowding distance)
  • Controle: Sistema fuzzy com 7 regras e superficie 3D
  • Economia: Inputs manuais com VPL, TIR, Monte Carlo
Para uso industrial: Calibrar os modelos fisicos com dados reais de planta (SCADA/PIMS), treinar a rede neural com dados historicos, e validar as faixas operacionais para o separador especifico.
